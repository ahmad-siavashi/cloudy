class documentation

class Vmm(ABC):

Known subclasses: policy.vmm.VmmSpaceShared

View In Hierarchy

This class provides a template for implementing virtual machine managers that can allocate and deallocate VMs on a host and process.

Method __contain__ Check if a Vm instance is present in the list of allocated VMs.
Method __iter__ Return an iterator for iterating over each Vm instance in the list of allocated VMs.
Method __len__ Return the number of guests managed by the host.
Method __post_init__ Undocumented
Method allocate The allocate function is responsible for allocating a new VM on the host. It takes as input a list of VMs and returns a list of booleans, one per VM, indicating whether it was successfully allocated.
Method deallocate The deallocate function is used to deallocate a list of VMs from the host. It returns a list of booleans, where each boolean indicates whether the VM was successfully deallocated or not.
Method has_capacity Check if the host has enough CPU, RAM, and GPU resources to allocate the given virtual machine.
Method idles The idles function returns list of guests that have no running load.
Method resume The resume function processes the guest VMs OS for the specified duration.
Constant HOST the physical machine on which the virtual machines managed by the Vmm instance are running
