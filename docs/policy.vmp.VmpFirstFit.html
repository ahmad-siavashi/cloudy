class VmpFirstFit(policy.Vmp):

View In Hierarchy

The VmpFirstFit class inherits from the abstract Vmp class and implements a first-fit VM placement algorithm. This algorithm attempts to allocate VMs to the first host that has enough resources to accommodate them.
