class OsTimeShared(policy.Os):

View In Hierarchy

This class is a subclass of the Os class and implements time-sharing scheduling.

Method resume The resume function takes a tuple of integers representing the number of cycles available on each CPU core and an integer representing the duration for which the process may run uninterrupted. It then runs the apps in a time-sharing scheduling, then returns the consumed cycles.
